# 🧘‍♂️ Bodify

O **Bodify** ajuda os usuários a calcular rapidamente seu **Índice de Massa Corporal (IMC)** e compreender seu estado de saúde por meio de recursos visuais intuitivos e interações amigáveis.

![BodifyDemo](https://github.com/KrishBharadwaj5678/Bodify/raw/main/BodifyDemo.jpeg)

## ✨ Funcionalidades

| Funcionalidade                        | Descrição                                                        |
| ------------------------------------- | ---------------------------------------------------------------- |
| 📊 Cálculo de IMC                     | Calcula o Índice de Massa Corporal com base na altura e no peso  |
| 🎚️ Controles Deslizantes Interativos | Entrada suave para altura e peso usando sliders                  |
| 🔄 Seleção de Gênero                  | Alternância animada utilizando um interruptor deslizante         |
| 📸 Captura de Tela                    | Permite capturar a tela de resultados do IMC                     |
| 🖼️ Salvar na Galeria                 | Exporta os resultados como imagens para a galeria do dispositivo |
| 🚀 Tela de Inicialização              | Tela de abertura animada ao iniciar o aplicativo                 |
| 📱 Interface Responsiva               | Funciona em diferentes tamanhos de tela                          |
| 🎨 UI/UX Moderna                      | Design limpo, minimalista e amigável ao usuário                  |
