# 🧘‍♂️ Bodify

Bodify helps users quickly calculate their Body Mass Index (BMI) and understand their health status with intuitive visuals and interactions. 

![BodifyDemo](https://github.com/KrishBharadwaj5678/Bodify/raw/main/BodifyDemo.jpeg)

## ✨ Features

| Feature                 | Description                                         |
| ----------------------- | --------------------------------------------------- |
| 📊 BMI Calculation      | Calculates Body Mass Index based on height & weight |
| 🎚️ Interactive Sliders | Smooth slider input for height and weight           |
| 🔄 Gender Selection     | Animated toggle using rolling switch                |
| 📸 Screenshot Capture   | Capture BMI result screen                           |
| 🖼️ Save to Gallery     | Export results as images to device gallery          |
| 🚀 Splash Screen        | Animated splash screen on app launch                |
| 📱 Responsive UI        | Works across different screen sizes                 |
| 🎨 Modern UI/UX         | Clean, minimal, and user-friendly design            |

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/KrishBharadwaj5678/Bodify.git
```

### 2️⃣ Navigate to project directory

```bash
cd Bodify
```

### 3️⃣ Install dependencies

```bash
flutter pub get
```

### 4️⃣ Run the app

```bash
flutter run
```
