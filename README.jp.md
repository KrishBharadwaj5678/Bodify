# 🧘‍♂️ Bodify

**Bodify** は、ユーザーが **BMI（Body Mass Index：体格指数）** を素早く計算し、直感的なビジュアルとインタラクションを通じて健康状態を理解できるよう支援するアプリです。

![BodifyDemo](https://github.com/KrishBharadwaj5678/Bodify/raw/main/BodifyDemo.jpeg)

## ✨ 主な機能

| 機能                | 説明                        |
| ----------------- | ------------------------- |
| 📊 BMI計算          | 身長と体重に基づいてBMIを計算          |
| 🎚️ インタラクティブスライダー | スムーズなスライダー操作で身長と体重を入力     |
| 🔄 性別選択           | ローリングスイッチによるアニメーション付き切り替え |
| 📸 スクリーンショット撮影    | BMI結果画面をキャプチャ             |
| 🖼️ ギャラリーに保存      | 結果を画像としてデバイスのギャラリーに保存     |
| 🚀 スプラッシュスクリーン    | アプリ起動時のアニメーション付きスプラッシュ画面  |
| 📱 レスポンシブUI       | さまざまな画面サイズに対応             |
| 🎨 モダンなUI/UX      | シンプルで洗練された使いやすいデザイン       |

## 🚀 はじめに

### 1️⃣ リポジトリをクローンする

```bash
git clone https://github.com/KrishBharadwaj5678/Bodify.git
```

### 2️⃣ プロジェクトディレクトリへ移動

```bash
cd Bodify
```

### 3️⃣ 依存関係をインストール

```bash
flutter pub get
```

### 4️⃣ アプリを実行

```bash
flutter run
```
