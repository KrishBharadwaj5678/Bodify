# 🧘‍♂️ Bodify

**Bodify** помогает пользователям быстро рассчитать **Индекс массы тела (ИМТ)** и понять своё состояние здоровья с помощью интуитивно понятной визуализации и удобного взаимодействия.

![BodifyDemo](https://github.com/KrishBharadwaj5678/Bodify/raw/main/BodifyDemo.jpeg)

## ✨ Возможности

| Возможность                | Описание                                                      |
| -------------------------- | ------------------------------------------------------------- |
| 📊 Расчёт ИМТ              | Вычисляет индекс массы тела на основе роста и веса            |
| 🎚️ Интерактивные ползунки | Удобный ввод роста и веса с помощью плавных слайдеров         |
| 🔄 Выбор пола              | Анимированный переключатель с эффектом прокрутки              |
| 📸 Создание скриншотов     | Возможность сделать снимок экрана с результатами ИМТ          |
| 🖼️ Сохранение в галерею   | Экспорт результатов в виде изображений в галерею устройства   |
| 🚀 Экран загрузки          | Анимированный экран приветствия при запуске приложения        |
| 📱 Адаптивный интерфейс    | Корректно работает на устройствах с разными размерами экранов |
| 🎨 Современный UI/UX       | Чистый, минималистичный и удобный дизайн                      |

## 🚀 Начало работы

### 1️⃣ Клонируйте репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/Bodify.git
```

### 2️⃣ Перейдите в директорию проекта

```bash
cd Bodify
```

### 3️⃣ Установите зависимости

```bash
flutter pub get
```

### 4️⃣ Запустите приложение

```bash
flutter run
```
